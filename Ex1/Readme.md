# Exercise 1 – Visual Studio Project Upload Instructions

These instructions apply **regardless of how you created your work**:
- Your own GitHub repository
- A repository created from a template
- A project that exists only on your local computer

---

## What You Must Upload
Upload your **Visual Studio 2022 project files** for this exercise.

Your submission must include:
- The `.sln` file (solution file), if applicable
- The project folder containing:
  - `.csproj`
  - `.cs` source file

## ⚠️ Important: Files to REMOVE Before Uploading
Before uploading, **delete these folders** if they exist:

- `bin/`
- `obj/`
- `.vs/`

These are automatically generated by Visual Studio and must NOT be submitted.

---

## Recommended Folder Structure
Your repository (or this folder) should look similar to:

Ex1/
├─ YourSolution.sln
├─ YourProject/
│ ├─ YourProject.csproj
│ ├─ Program.cs
│ └─ other .cs files

## How to Upload Using GitHub (No Git Required)

### Step 1: Open Your GitHub Repository
- Go to https://github.com
- Open the repository you are using for this exercise
  - (Create a new public repository if you do not have one)

---

### Step 2: Navigate to the Correct Folder
- Open the `Ex1` folder (or the main repository folder if no `Ex1` folder exists)

---

### Step 3: Upload Your Files
1. Click **Add file → Upload files**
2. Drag and drop:
   - Your `.sln` file
   - Your project folder
   - Any required documents
3. Click **Commit changes** 

---

## If Your Work Is Only on Your Computer
1. Locate your Visual Studio project folder on your computer
2. Delete `bin/`, `obj/`, and `.vs/` folders
3. Follow the upload steps above
